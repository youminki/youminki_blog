
#### 🤖 **자동 동기화 시스템**

이 프로젝트는 **Velog 블로그와 자동으로 동기화**되는 시스템을 포함하고 있습니다:

- **RSS 피드 활용**: Velog RSS를 통해 최신 포스트 자동 수집
- **GitHub Actions**: 매일 자동으로 동기화 실행
- **중복 제거**: 기존 포스트와 중복되지 않는 새 포스트만 추가
- **카테고리 자동 분류**: 제목 기반으로 카테고리 자동 할당
- **실시간 업데이트**: Velog에 새 글을 올리면 자동으로 포트폴리오에 반영

#### 📝 **동기화 설정**

1. **자동 동기화**: GitHub Actions가 매일 오전 9시(한국 시간)에 실행
2. **수동 동기화**: `yarn sync-velog` 명령어로 언제든 실행 가능
3. **카테고리 매핑**: 제목에 포함된 키워드로 자동 카테고리 분류
   - "React" → React 카테고리
   - "이슈", "문제", "해결" → 경험했던 이슈 카테고리
   - "성능", "최적화" → Performance 카테고리
   - 기타 → Programming 카테고리
